ShapeDetector — Класифікатор геометричних фігур на TensorFlow.js

ShapeDetector — це веб-застосунок, який визначає тип геометричної фігури на зображенні за допомогою згорткової нейронної мережі, що працює повністю у браузері. Підтримуються три класи: коло, квадрат і трикутник.

ОСНОВНИЙ ФУНКЦІОНАЛ:
////////////////////////////////////////////

Класифікація зображень у реальному часі.

Завантаження файлів через інтерфейс.

Автоматичне масштабування та попередня обробка зображення.

Виведення ймовірностей для кожного класу.

Відображення інформації про файл: розмір, роздільність, формат.

Можливість збереження зображення у PNG, JPG та BMP.

Модель працює повністю локально, без сервера.
